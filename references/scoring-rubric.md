# Scoring Rubric — SEO / AEO / GEO Audit

## Section Weights for Overall Score

| Section | Weight |
|---------|--------|
| Technical SEO | 15% |
| On-Page SEO | 15% |
| Website Speed & Core Web Vitals | 10% |
| Content Quality & E-E-A-T | 15% |
| Backlink Profile | 10% |
| Competitor Analysis | 5% (informational, light weight) |
| Local SEO | 10% (redistribute to other sections if not applicable) |
| AEO | 10% |
| GEO | 10% |
| Mobile-Friendliness & UX | 5% |

---

## Section Scoring Criteria

### 1. Technical SEO (0–100)

| Criteria | Points |
|----------|--------|
| HTTPS enabled | 10 |
| XML sitemap present and submitted | 10 |
| robots.txt present and correct | 10 |
| No crawl errors (4xx/5xx visible) | 15 |
| Canonical tags in use | 10 |
| Structured data / schema present | 15 |
| Clean URL structure | 10 |
| No duplicate content signals | 10 |
| Hreflang (if multi-language) | 5 |
| Site indexed correctly (site: query) | 5 |

---

### 2. On-Page SEO (0–100)

| Criteria | Points |
|----------|--------|
| Title tag present, keyword-rich, ≤60 chars | 15 |
| Meta description present, compelling, ≤160 chars | 10 |
| H1 present and unique | 10 |
| H2/H3 structure logical | 10 |
| Keyword used naturally in first 100 words | 10 |
| Images have alt text | 10 |
| Internal linking present | 10 |
| Outbound links to authoritative sources | 5 |
| FAQ or structured Q&A content | 10 |
| Open Graph / Twitter Card tags | 10 |

---

### 3. Website Speed & Core Web Vitals (0–100)

| Criteria | Points |
|----------|--------|
| LCP (Largest Contentful Paint) < 2.5s | 25 |
| FID/INP < 100ms | 20 |
| CLS < 0.1 | 20 |
| Page size < 3MB | 10 |
| Uses CDN | 10 |
| Images in modern format (WebP/AVIF) | 10 |
| Minified CSS/JS | 5 |

*If CWV data unavailable, score based on available signals and note manual check needed.*

---

### 4. Content Quality & E-E-A-T (0–100)

| Criteria | Points |
|----------|--------|
| **Experience** — First-hand content, case studies, reviews | 15 |
| **Expertise** — Author bios, credentials, qualifications | 20 |
| **Authoritativeness** — Brand mentions, press, awards | 20 |
| **Trustworthiness** — HTTPS, privacy policy, contact page, about page | 20 |
| Content depth and word count appropriate for topic | 10 |
| Content freshness / recently updated | 10 |
| No thin or duplicate content | 5 |

---

### 5. Backlink Profile (0–100)

| Criteria | Points |
|----------|--------|
| Domain Authority / DR signal (high = good) | 30 |
| Number of referring domains | 25 |
| Quality of linking sites (news, edu, gov = high) | 25 |
| No toxic/spammy backlinks evident | 20 |

*Score relatively — a local plumber with 20 good local links scores higher than a national brand with 20 links.*

---

### 6. Competitor Analysis (0–100)

This section is primarily informational. Score based on:

| Criteria | Points |
|----------|--------|
| Able to identify 10 clear competitors | 30 |
| Site has visible differentiators vs competitors | 40 |
| Site is competitive on visible SEO signals | 30 |

---

### 7. Local SEO (0–100) — Skip if not applicable

| Criteria | Points |
|----------|--------|
| Google Business Profile claimed & verified | 25 |
| NAP consistent across web | 20 |
| Local citations present (Yelp, YellowPages, etc.) | 15 |
| Reviews present and responded to | 15 |
| LocalBusiness schema on site | 15 |
| Location pages or city landing pages present | 10 |

---

### 8. AEO — Answer Engine Optimization (0–100)

| Criteria | Points |
|----------|--------|
| FAQ sections present on key pages | 20 |
| FAQ schema markup implemented | 20 |
| Content answers specific "who/what/when/where/why/how" questions | 20 |
| HowTo schema present (if relevant) | 10 |
| Speakable schema present | 10 |
| Content formatted for featured snippet capture (clear definitions, lists) | 20 |

---

### 9. GEO — Generative Engine Optimization (0–100)

| Criteria | Points |
|----------|--------|
| Brand entity clearly defined on site (name, location, services, founding) | 20 |
| Clear, factual, citable content (statistics, dates, specifics) | 20 |
| Topical authority — multiple deep pages on core topic | 20 |
| Wikipedia / knowledge panel presence | 10 |
| Mentions on authoritative third-party sites (press, directories) | 15 |
| Content depth appropriate for AI training/referencing | 15 |

---

### 10. Mobile-Friendliness & UX (0–100)

| Criteria | Points |
|----------|--------|
| Responsive viewport meta tag present | 20 |
| Text readable without zooming | 20 |
| Tap targets appropriately sized | 15 |
| No horizontal scrolling on mobile | 15 |
| Fast mobile load (related to CWV) | 15 |
| Clear navigation and CTA on mobile | 15 |

---

## Grade Thresholds

| Score | Grade | Meaning |
|-------|-------|---------|
| 90–100 | A+ | Excellent — leading practice |
| 80–89 | A | Strong — minor improvements only |
| 70–79 | B | Good — some gaps to address |
| 60–69 | C | Fair — meaningful work needed |
| 50–59 | D | Poor — significant issues |
| Below 50 | F | Critical — urgent action required |
